Overview over my offline tools

  • XMLWPFToolbox (.NET Windows app allowing the use of XSLT 3.0, XQuery 3.1, XPath 3.1 with Saxon HE and XSD schema 1.0/1.1 with Apache Xerces)
  • Notepad++ xmltools fork providing XSLT 3.0 support using SaxonC HE 12 (in addition to the existing xmltools features)>

  • dotnet tools/.NET command line tools

    • SaxonHE12NetXslt: XSLT 3.0 from the command line (Saxon HE 12 Java cross-compiled with IKVM)
    • SaxonHE12NetXQuery: XQuery 3.1 from the command line (Saxon HE 12 Java cross-compiled with IKVM)
    • SaxonHE12NetIXslt: XSLT 3.0 with iXML support from the command line (Saxon HE 12 Java and Coffeesacks cross-compiled with IKVM)
    • SaxonHE12NetIXQuery: XQuery 3.1 with iXML from the command line (Saxon HE 12 Java and Coffeesacks cross-compiled with IKVM)
